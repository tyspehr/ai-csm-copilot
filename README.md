# AI Customer Success Risk Copilot

AI-powered customer success dashboard built with Python, Streamlit, and OpenAI APIs.

## Overview

This project simulates a real-world enterprise customer success workflow by analyzing customer account health, identifying churn risk indicators, summarizing account sentiment, and generating AI-powered executive recommendations and follow-up actions.

The application was designed to demonstrate how large language models can support customer success, account management, RevOps, and healthcare SaaS workflows.

---

## Features

- Customer account health analysis
- Rule-based churn risk scoring
- AI-generated executive summaries
- AI-powered customer sentiment analysis
- Recommended retention actions
- Draft customer follow-up emails
- Interactive Streamlit dashboard
- Simulated enterprise healthcare/GPO account dataset

## Local Installation

### Clone Repository

```bash
git clone <your-repo-url>
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Add OpenAI API Key

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
```

### Run Application

```bash
python -m streamlit run app.py
```
